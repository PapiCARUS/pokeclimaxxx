## Card collection

- Il existe moins de dépôts open source matures avec une UI aussi solide que Collectr, TCGplayer ou PriceCharting, qui sont principalement produits propriétaires.
- Référence open source utile à chercher ensuite : projets `pokemon-tcg`, `tcg collection tracker`, `card collection manager` sur GitHub, en privilégiant ceux avec captures récentes et activité réelle.
- Attention : ne pas copier les assets de cartes, logos, images ou bases propriétaires.
