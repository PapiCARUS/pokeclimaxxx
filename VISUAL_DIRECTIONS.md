# Visual Directions — ERP Climax

## Direction 1 — B2B dense et sobre

- Références : Linear, GitHub, Retool, Supabase, ERPNext.
- Palette : gris froids, blanc cassé, accent bleu/indigo discret, statuts codifiés.
- Typographie : Inter, Geist, IBM Plex Sans ou system sans.
- Densité : élevée.
- Cartes : sobres, bordure fine, ombre minimale.
- Tableaux : très centraux, lignes compactes, colonnes fixes.
- Boutons : rectangulaires, 6-8 px radius, action primaire nette.
- Navigation : sidebar groupée, top bar utilitaire.
- Responsive : desktop prioritaire, mobile repensé.
- Avantages : sérieux, efficace, crédible ERP.
- Limites : risque de froideur.
- Compatibilité cible : forte pour vendeurs qui gèrent beaucoup de références.
- Risque générique : moyen si trop Linear.
- Complexité : moyenne.

## Direction 2 — Financier premium opérationnel

- Références : Stripe, Ramp, Brex, Pennylane, Revolut Business, Wise Business.
- Palette : neutres premium, bleu nuit/vert finance, touches amber pour alertes.
- Typographie : Inter, Suisse-like, system sans.
- Densité : moyenne-haute.
- Cartes : KPI fortes, montants très lisibles, deltas nets.
- Tableaux : transactions, dépenses, ventes, montants alignés.
- Boutons : sobres, hiérarchie claire.
- Navigation : très propre, moins colorée.
- Responsive : excellent potentiel mobile.
- Avantages : confiance, sérieux, revenus/dépenses valorisés.
- Limites : peut masquer le côté stock/cartes.
- Compatibilité cible : forte si Climax veut paraître professionnel et fiable.
- Risque générique : moyen-élevé si trop fintech.
- Complexité : moyenne.

## Direction 3 — E-commerce opérationnel

- Références : Shopify Admin, Square, Lightspeed, Shopify POS, eBay Seller Hub.
- Palette : gris chauds, accent vert/bleu, statuts commerce.
- Typographie : system sans robuste.
- Densité : moyenne-haute.
- Cartes : orientées actions et alertes.
- Tableaux : produits, commandes, ventes, statuts.
- Boutons : action rapide visible.
- Navigation : modules vendeur.
- Responsive : mobile très actionnable.
- Avantages : proche du quotidien des vendeurs.
- Limites : moins distinctif si trop Shopify.
- Compatibilité cible : très forte.
- Risque générique : moyen.
- Complexité : moyenne.

## Direction 4 — SaaS moderne mais pragmatique

- Références : Vercel, Notion, Airtable, Asana, Monday.
- Palette : blanc, gris neutres, accents colorés maîtrisés.
- Typographie : Inter/Geist.
- Densité : moyenne.
- Cartes : modulaires.
- Tableaux : propres mais moins compacts.
- Boutons : modernes, arrondis modérés.
- Navigation : légère.
- Responsive : facile à adapter.
- Avantages : accessible, moderne.
- Limites : risque de reproduire le défaut actuel : trop vide, trop générique, trop IA.
- Compatibilité cible : moyenne.
- Risque générique : élevé.
- Complexité : faible à moyenne.

## Direction 5 — Hybride ERP + marketplace cartes

- Références : Shopify Admin, Cardmarket, TCGplayer, Collectr, Pokécardex, PriceCharting, Whatnot, Vinted.
- Palette : neutres professionnels + accent collection discret, badges état/rareté.
- Typographie : Inter ou IBM Plex Sans.
- Densité : élevée sur tables, moyenne sur galeries.
- Cartes : visuels cartes enrichis de prix, état, quantité.
- Tableaux : stock/ventes très denses.
- Boutons : actions vendeur rapides.
- Navigation : modules ERP, mais pages cartes plus visuelles.
- Responsive : mobile très fort pour scan/galerie/vente.
- Avantages : identité propre à Climax, moins générique.
- Limites : demande un système visuel plus précis.
- Compatibilité cible : très forte.
- Risque générique : faible si bien exécuté.
- Complexité : élevée.

## Direction recommandée

Direction principale recommandée : **Hybride ERP + marketplace cartes**, avec base **B2B dense et sobre** et accents **e-commerce opérationnel**.

Motif : Climax ne doit pas ressembler à un template SaaS abstrait. Le public cible manipule stock, cartes, prix, ventes et plateformes. L’interface doit donc être professionnelle, dense, vendeuse et visuelle seulement quand la carte/produit l’exige.

## Décisions visuelles à valider

- Mode clair uniquement au départ ou clair + sombre ?
- Accent couleur principal : bleu professionnel, vert commerce, violet discret ou rouge/bleu cartes ?
- Niveau d’identité collection : subtil ou assumé ?
- Densité par défaut : compacte ou confortable ?
- Galerie cartes prioritaire dans Stock ou uniquement Base cartes ?
